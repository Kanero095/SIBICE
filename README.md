## About Projek

**SIBICE (Sriwijaya Business Center)** merupakan program inkubator bisnis bagi UMKM binaan Dinas Koperasi dan UKM Provinsi Sumatera Selatan yang berfokus pada pengembangan dan akselerasi usaha rintisan.

Sebagai sebuah inkubator bisnis, SIBICE menyediakan ekosistem terintegrasi berupa pendampingan, pelatihan, mentoring, serta akses terhadap fasilitas dan jejaring bisnis. Program ini bertujuan untuk meningkatkan kapabilitas pelaku UMKM dalam mengembangkan ide bisnis, memvalidasi model usaha, hingga meningkatkan daya saing di pasar yang lebih luas.

Melalui proyek aplikasi web ini, proses manajemen, pendaftaran, dan pembinaan UMKM di lingkungan SIBICE didigitalisasi agar menjadi lebih efisien, terstruktur, dan mudah diakses oleh seluruh pemangku kepentingan.

-   **Sistem Registrasi UMKM Digital**
    Memungkinkan pelaku UMKM melakukan pendaftaran secara online tanpa proses manual atau dokumen fisik yang berlebihan.

-   **Manajemen Data UMKM Terpusat**
    Admin dapat mengelola, memverifikasi, memperbarui, dan mengarsipkan data UMKM secara terstruktur dalam satu dashboard terintegrasi.

-   **Sistem Autentikasi Aman**
    Dilengkapi fitur login dengan verifikasi email serta dukungan pengembangan Single Sign-On (SSO) berbasis OAuth 2.0.

-   **Dashboard Monitoring & Statistik**
    Menampilkan data visual terkait jumlah UMKM terdaftar, status pembinaan, dan perkembangan program inkubasi.

-   **Manajemen Program Inkubasi**
    Mengatur tahapan inkubasi mulai dari seleksi, pendampingan, pelatihan, hingga evaluasi perkembangan UMKM.

-   **Upload & Manajemen Dokumen**
    Mendukung unggah file (legalitas usaha, proposal, laporan, dll) secara aman dan terorganisir.

-   **Notifikasi & Status Progres**
    UMKM dapat memantau status pengajuan dan perkembangan program secara real-time.

-   **Role-Based Access Control (RBAC)**
    Pembagian hak akses sesuai peran (Admin, Mentor, UMKM) untuk menjaga keamanan dan integritas data.

-   **Responsive & User-Friendly Interface**
    Desain antarmuka adaptif dan ramah pengguna di berbagai perangkat (desktop, tablet, smartphone).
